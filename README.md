SETU – Donation Platform
SETU is a MERN-based donation platform that connects donors, charities, and communities in a single system.

It supports both monetary and non-monetary donations with transparency, real-time tracking, and donor recognition features.

🚀 Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
🔑 Core Features
One-time & recurring donations
Monetary + goods donation support
Campaign creation
Team-based fundraising
Hall of Fame (Top donors)
Real-time donation tracking
Commenting & reporting system
Role-based dashboards (Donor/Admin)
